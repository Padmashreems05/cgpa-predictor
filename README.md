# 🎓 CGPA Predictor

I built this project to help track and predict student CGPA for the Engineering 2023–27 batch.

You enter semester-wise SGPA marks for each student and the app automatically calculates their CGPA and predicts how they will perform in future semesters.

## What it does
- Shows 45 students across 8 branches (CSE, ECE, MECH, CIVIL etc.)
- Add new students and enter their SGPA
- See CGPA growth charts and semester performance graphs
- Predicts future CGPA using a simple math formula
- Search students by name or filter by branch

## How to run it
Download the code, open terminal in the folder and run:
npm install
npm run dev
Then open http://localhost:5173 in your browser.

## Made with
React, Recharts, Vite
